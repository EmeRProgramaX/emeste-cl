# EMESTE - Hoja de ruta de marca, web y crecimiento

## Vision general

EMESTE debe funcionar como una marca paraguas premium que une tecnologia, musica y mentorias con un mismo proposito: construir soluciones, experiencias y contenidos que ayuden a personas y negocios a crecer con orden, conciencia, respeto y alegria.

## Divisiones principales

### 1. EMESTE Software
Servicios principales:
- Desarrollo web premium.
- Landing pages comerciales.
- Software a medida.
- Sistemas POS, ERP, inventario y ventas.
- Automatizaciones.
- Integraciones con IA.
- Consultoria tecnologica.
- Soporte informatico.

Objetivo inmediato:
Generar liquidez vendiendo servicios claros, rentables y rapidos de implementar.

### 2. EMESTE DJ
Objetivo:
Construir una presencia artistica seria, moderna y profesional bajo el nombre EMESTE.

Contenidos sugeridos:
- Press kit artistico.
- Biografia profesional.
- Fotos oficiales.
- Videos de presentaciones.
- Mixes y sesiones.
- Booking para eventos.
- Integracion con Instagram, YouTube y WhatsApp.

### 3. EMESTE Mentorias
Objetivo:
Crear contenido educativo de crecimiento personal, vida en paz, amor, respeto, solidaridad y conciencia.

Contenidos sugeridos:
- Videos educativos.
- Reflexiones.
- Mentorías individuales o grupales.
- Blog o biblioteca de aprendizajes.
- Canal YouTube: Emeste Mente Diferente.

## Arquitectura recomendada para emeste.cl

Rutas principales:
- `/` Inicio corporativo premium.
- `/software` Servicios de programacion, automatizacion y consultoria.
- `/dj` Carrera artistica, press kit y booking.
- `/mentorias` Emeste Mentorias y contenido educativo.
- `/actualizaciones` Noticias, videos, lanzamientos y novedades.
- `/contacto` WhatsApp, agenda y formulario.

## Prioridad comercial

1. Mantener la pagina principal enfocada en vender software y servicios tecnologicos, porque es la linea con retorno economico mas inmediato.
2. Agregar DJ y mentorias como divisiones profesionales sin diluir el mensaje principal.
3. Crear llamadas a la accion separadas:
   - Cotizar software.
   - Contratar DJ.
   - Ver mentorias.
4. Centralizar todas las redes en emeste.cl para que la marca crezca como ecosistema.

## Requisitos tecnicos

- Next.js con App Router.
- Tailwind CSS.
- Framer Motion para animaciones profesionales.
- GitHub como fuente de verdad.
- Vercel para despliegue automatico.
- Secciones modulares y escalables.
- SEO tecnico por division.
- Datos de contacto visibles: WhatsApp, correo, calendario e Instagram.
